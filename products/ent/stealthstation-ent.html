StealthStation™ ENT Navigation System

The StealthStation™ ENT navigation system uses electromagnetic technology to enable image-guided surgery during ear, nose, and throat procedures.

Description

StealthStation™ ENT Medtronic’s most advanced ENT navigation system

  • Two different electromagnetic (EM) emitter options provide more flexibility with surgical setup
  • Manage multiple navigated instruments simultaneously
  • Add-on software modules for enhanced visualization and surgical planning

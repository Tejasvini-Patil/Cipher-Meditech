Product Overview

The IPC® (Integrated Power Console) System is the central command unit for Medtronic's family of ENT powered surgical instruments. This advanced console provides precise motor control, irrigation management, and an intuitive user interface for streamlined surgical workflow.

Compatible with the full range of Medtronic ENT handpieces and blades, the IPC System enables surgeons to quickly configure settings for different procedures and save preferred parameters for efficient setup in future cases.

Key Features

  • Touchscreen control interface
  • Dual handpiece ports
  • Integrated irrigation pump
  • Adjustable flow rate control
  • Preset memory for surgical preferences
  • Real-time speed and mode display
  • Multiple oscillation modes
  • Compact, mobile cart design

Compatible Instruments

  • Diego® Elite Multidebrider
  • Straightshot® M5 Microdebrider
  • Tricut® Blade System
  • XPS® Microdebrider family
  • All Medtronic ENT powered handpieces

Technical Specifications

Handpiece Ports 2 channels
Speed Control Variable 0-15,000 RPM

Irrigation Integrated pump
Flow Rate 0-200 ml/min
